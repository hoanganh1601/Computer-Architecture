# Computer Architecture - PTIT 

## Description
This repository contains my personal collection of Assembly (`.asm`) programs written for the Computer Architecture course at the Posts and Telecommunications Institute of Technology (PTIT). These files serve as a practical archive for exam revision, basic algorithmic problem-solving, and understanding low-level computer operations.

## File Overview
Here are some of the key assembly exercises included in this project:

* **`HelloWorld.asm`**: A basic program to print a greeting to the console.
* **`tong2so.asm`**: A program that calculates the sum of two numbers.
* **`in_chuoi.asm`**: Demonstrates how to print strings.
* **`base_loop.asm`**: Examples of basic loop structures in Assembly.
* **`checkSoHopLe(d24 - ky 1).asm`**: Exam revision code for validating numerical inputs.
* **`codeNhietDo.asm`**: A program for temperature-related calculations.
* **`in_nghich_dao.asm`**: Reverses the input and prints it.

## Software Requirements
To run these files, you will need an 8086 microprocessor emulator. This project uses **EMU8086**.

## How to Download and Run

**Step 1: Download the Code**
1. Click on the green `<> Code` button at the top of this repository.
2. Select **Download ZIP** and extract the folder to your computer, or clone the repository using Git:
```bash
   git clone [https://github.com/hoanganh1601/Computer-Architecture.git](https://github.com/hoanganh1601/Computer-Architecture.git)

**Step 2: Install EMU8086**
1. Download the EMU8086 software from your course materials or a trusted educational source.
2. Follow the standard installation prompts to install it on your Windows machine.

**Step 3: Run the Assembly Files**
1. Open the **EMU8086** application.
2. Click on the **Open** button in the top menu.
3. Navigate to the folder where you saved this repository and select an `.asm` file (for example, `HelloWorld.asm`).
4. Click the **Emulate** button (the green play icon) at the top of the screen.
5. A new window will appear. Click **Run** in the emulator window to execute the code and view the results!
